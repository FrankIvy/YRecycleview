##这是一个下拉刷新和下拉加载的Recycleview的控件,实现的想法来自于XListview
使用方法很简单
```javascript
    布局文件添加
     <yuan.kuo.yu.view.YRecycleview
        android:id="@+id/ycl"
        android:layout_width="match_parent"
        android:layout_height="match_parent" />



```
###其他的适配器以及正常编写都是和官方的一样,没有任何改变上手比较容易简单
